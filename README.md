# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Clean, professional layout
- Sections for About, Projects, and Achievements
- Easy to customize

## Setup

1. Clone or download this repository
2. Customize the content in `index.html`
3. Add your images:
   - `pfp.jpeg` - Your profile picture (should be square/circular crop friendly)
   - `project.jpg` - An image representing your projects
4. Add your resume as `resume.pdf`
5. Update the styles in `styles.css` if needed

## Customization

### Personal Information
- Replace `[Your Name]` with your actual name
- Update the about section text
- Add your real projects and achievements

### Styling
- Colors can be changed in `styles.css`
- Font can be updated in the body selector
- Layout can be modified by adjusting the CSS grid/flexbox properties

### Images
- Profile picture should be 200x200px or larger (will be cropped to circle)
- Project image can be any size (will be responsive)

## Deployment

### GitHub Pages
1. Push your changes to a GitHub repository
2. Go to repository Settings > Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Your site will be available at `https://yourusername.github.io/repository-name`

### Other Hosting
- Upload the files to any web hosting service
- Ensure the server supports HTML, CSS, and JS files
